
#  Prime Factor TDD Kata

## Introduction
This is a simple TDD kata on prime factor. The objective of this kata is to practice TDD and refactoring.

## Usage
1. Clone the repository
2. Run `npm install`
3. Run `npm run build`
4. Run `npm run start`
